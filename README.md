# Privacy Policy

**Last updated:** June 14, 2026

## Overview

This Privacy Policy describes how this mobile game ("the Game") handles information. The Game is a single-player arcade game developed with Unity Engine.

---

## Information We Collect

**We do not collect, store, or transmit any personal information.**

The Game operates entirely offline. No data is sent to any server or third party.

### Data Stored Locally on Your Device

The Game saves the following data **only on your device** using Unity PlayerPrefs:

| Data | Purpose |
|------|---------|
| Best Score (`HelixJump_BestScore`) | Display your high score between sessions |

This data never leaves your device and can be cleared by uninstalling the game.

---

## Internet Access

The Game **does not require** and **does not use** an internet connection. No network requests are made at any time.

---

## Third-Party Services

The Game does **not** integrate any third-party services, including:

- No advertising networks (e.g., AdMob, Unity Ads)
- No analytics services (e.g., Firebase, Unity Analytics)
- No social media SDKs
- No crash reporting tools
- No in-app purchase services

---

## Children's Privacy

This Game does not collect any data from anyone, including children under the age of 13. It is safe to use by all ages in accordance with COPPA (Children's Online Privacy Protection Act) and similar regulations.

---

## Permissions

The Game does not request any sensitive Android permissions. It does not access:

- Camera
- Microphone
- Location
- Contacts
- Files or storage (beyond its own app data)

---

## Changes to This Policy

If the Game is updated to include features that collect data (e.g., ads, analytics), this Privacy Policy will be updated accordingly before such a version is published.

---

## Contact

If you have any questions about this Privacy Policy, you can contact the developer via the app store listing page where you downloaded this game.
